# ✨Game Story
> 쿠키런과 비슷한 1인 Play 게임이다.<br><br>
> 주인공 포테이토는 웨지감자가 되지 않기 위해 에어프라이어를 탈출해야 한다.<br><br>
> 에어프라이어 내부에는 혹시 포테이토가 탈출할까 걱정이 되었던 인간이 넣어둔 장애물들과 괴물들이 있다.<br><br>
> 이 장애물들과 괴물들의 공격을 피하며 에어프라이어를 탈출하는 게임이다.
